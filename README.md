# farm-front

### Install

Follow the following steps to get development environment running.

* Clone _'farm-front'_ repository from GitHub

  ```bash
  git clone git@github.com:antbaobao/farm-front.git
  ```

* Install node modules

   ```bash
   cd farm-front
   npm install
   ```

### Run

#### Run Start

    ```bash
    npm start
    visit localhost:3000
    ```
